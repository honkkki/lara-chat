# lara-chat
:rainbow:
🎨

- build with laravel/workerman

- 利用websocket实现的简单聊天室，支持多房间聊天和私聊。

- 安装组件
```
composer install
npm install
```

- 数据库文件在根目录sql中

- 进入socket下GatewayWorker的文件夹启动
```
php start.php start
```

- 项目目录启动
```
npm run watch-poll
```
